It's Tuesday, Odri.

Today we will do some chest exercises.

Before and after every workout, it is necessary to stretch each muscle. This will help prevent cramps and injuries. I have selected a few exercises that we can start with.
Don't rush. Take all the time you need to do the exercise correctly. Today we are not chasing a number, but rather the correct execution of the exercises.

The first video, which is before the exercises themselves, is about warming up the body. It is only a few minutes, but really important minutes.
